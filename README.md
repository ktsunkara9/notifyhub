# NotifyHub - Notification System

A comprehensive notification system built with Java and Spring Boot.

## Getting Started

### Prerequisites
- Java 17 or higher
- Maven 3.6 or higher

### Running the Application
```bash
mvn spring-boot:run
```

The application will start on `http://localhost:8080`

### Building the Project
```bash
mvn clean install
```
